Para activar tu tutor inteligente necesitas una API Key de Groq — completamente gratis y sin tarjeta de crédito.

Cómo obtener tu key gratis:
1. Ve a console.groq.com
2. Crea una cuenta (sin tarjeta de crédito)
3. Toca API Keys → Create API Key
4. Copia la key y pégala aquí abajo 👇
🔒 Tu key se guarda solo en este navegador. Nadie más puede verla.
